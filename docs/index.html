proton-mail-mcp

An MCP server for Proton Mail — send, read, search, and organize email from any AI client.

View on GitHub
30+ tools · SMTP + IMAP · Proton Bridge native · MIT licensed

Send, reply, forward

With threading headers and attachment passthrough.

Search & threads

By sender, subject, body, date, attachments; cross-folder thread walks.

Bulk organize

Move, delete, flag, and label hundreds of messages with dry-run previews.

Labels & folders

Create, rename, delete; full Proton label semantics.

Drafts

Save to \Drafts via IMAP APPEND, ready to send from the Proton client.

Safe by default

READONLY mode, sanitized HTML, BCC masking, validated inputs.

Install

Add to your MCP client config:

{
  "mcpServers": {
    "protonmail": {
      "command": "node",
      "args": ["/path/to/proton-mail-mcp/build/index.js"]
    }
  }
}

Environment variables:

PROTONMAIL_USERNAME=your-email@pm.me
PROTONMAIL_PASSWORD=your-smtp-password
PROTONMAIL_HOST=smtp.protonmail.ch
PROTONMAIL_PORT=587
IMAP_HOST=127.0.0.1
IMAP_PORT=1143
